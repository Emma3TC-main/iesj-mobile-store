

# 📱 iESJ Mobile Store

## 📌 Resumen del Proyecto

**iESJ Mobile Store** es una aplicación móvil orientada al comercio electrónico de hardware computacional, desarrollada bajo un enfoque moderno basado en arquitectura cliente-servidor y consumo de servicios REST.

El proyecto adopta un enfoque **mobile-first**, priorizando la experiencia del usuario en dispositivos móviles mediante interfaces intuitivas, navegación fluida y tiempos de respuesta eficientes.

La solución integra:

* 📲 **Frontend:** React Native (Expo)
* 🌐 **Backend:** API REST desacoplada (Node.js + Express)
* 🗄️ **Base de datos:** Sistema relacional
* 🔐 **Seguridad:** Autenticación con JWT

Este enfoque permite una clara separación de responsabilidades, facilitando la **escalabilidad**, **mantenibilidad** y evolución del sistema.
